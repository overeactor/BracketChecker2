Detailed Description

Unit tests for BracketChecker2 using Google Test.

This file contains test cases to verify the correctness of the BracketChecker2 library. It covers:

  • Bracket matching and mismatching scenarios.
  • Detection of formatting violations (long lines, macros).
  • Ignoring brackets inside strings and comments.
  • File reading behavior and edge cases (empty input, comments only).

Each test checks the output of parse_brackets or code_validation against expected BracketError results.
