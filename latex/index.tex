This project is a simple bracket validation tool that scans a text file and verifies that all brackets (parentheses, square brackets, and curly braces) are correctly matched.\hypertarget{index_usage}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage}

\begin{DoxyItemize}
\item The program reads an input file containing code or text.
\item It detects any unmatched opening or closing brackets.
\item The results are written to an output file.
